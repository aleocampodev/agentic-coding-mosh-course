# Agentic Coding with Mosh

## Projects

### expense-tracker-starter
React expense tracker app.

```bash
cd expense-tracker-starter
npm install
npm run dev
```

## Architecture
- Each project is self-contained
- React + Vite for frontend projects
- No TypeScript, no test framework (unless specified)

## Notes
- ESLint `no-unused-vars` rule ignores vars starting with `[A-Z_]`
- Run `npm run lint` before committing
