# Agentic Coding with Mosh

A collection of hands-on projects to master AI-assisted software development using Claude Code and other AI tools.

## Projects

### 1. Expense Tracker (Starter)
A React-based expense tracking app to learn the fundamentals.

**Tech Stack:** React, Vite, Recharts

**Getting Started:**
```bash
cd projects/expense-tracker-starter
npm install
npm run dev
```

## About This Course

Learn how to leverage AI agents to:
- Write cleaner, more efficient code
- Debug issues faster
- Refactor with confidence
- Build production-ready applications

Each project includes intentional bugs and challenges to solve using AI-assisted techniques.

## Requirements

- Node.js 18+
- npm or pnpm
- Claude Code (or similar AI assistant)
